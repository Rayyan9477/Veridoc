The Verification Society

Every extractor hands you JSON from one model and asks you to trust it. Veridoc runs a society of models — an Extractor and an Auditor read the page independently, a Reconciler arbitrates, and a Critic breaks ties, each on a different Qwen model — so every field is cross-examined, grounded to the source pixel, and shipped with calibrated confidence.

Validated live on Qwen Cloud — 21 fields extracted from a real invoice, 3 distinct models cross-examining, provenance + calibrated confidence emitted.
